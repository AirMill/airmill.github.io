FAQ

Short answers for normal users.

Where do I download the app?

Use the Download (Latest) button — it opens the app’s GitHub Releases page. Download the file for your system (Windows .exe, macOS .dmg/.zip if available).

Windows shows a warning (“protected your PC”). Is it safe?

This is common for apps without a paid code-signing certificate. Download only from the official Releases page on this GitHub account. On the warning screen choose More info → Run anyway.

Do these apps upload my files?

No. These tools are designed to run locally. Your files stay on your computer.

For contributors

If you’re technical.

How do I report bugs or request features?

Open an issue on the specific app repository and include: OS version, app version, steps to reproduce, and screenshots if possible.

Can I translate the UI?

Yes — open an issue and we can add a simple translations file per app.

What’s the roadmap?

Cleaner UI, better installers, and more photographer-focused workflow tools.
